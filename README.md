# Memory
다양한 라이브러리와 오픈소스 활용 능력 향상을 위한 프로젝트

<h3>목표</h3>
-기억에 남는 장소와 추억을 사진으로 남겨 위치나 자세한 내용을 저장하여 목록 또는 지도, 달력 등에 표시하여 저장하거나 공유할 수 있도록 함

<h3>라이브러리 및 오픈소스</h3>
